В данном проекте реализован свой сервис для генерацию API 
Инструкция по разворачиванию: 
Нам потребуется phpMyAdmin там создаем нашу базу под названием api в этой базе создаем таблицу myapi которая содержит в себе 
два столбца первый столбец id тип int с AUTO_INCREMENT второй столбец numbApi тип varchar
Для подключения к базе все стандартно
При каждом обращении мы генерируем значение (api ключ) они записываются поочередно.
